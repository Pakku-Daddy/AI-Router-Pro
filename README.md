# AI Router Pro

A universal AI router that intelligently routes prompts to the best AI provider.

## Features

- Gemini
- OpenRouter
- Groq
- OpenAI
- NVIDIA
- Cerebras
- Automatic provider routing
- Conversation memory
- Streaming responses
- CLI interface

## Installation

```bash
git clone https://github.com/Pakku-Daddy/AI-Router-Pro.git
cd AI-Router-Pro

python -m venv .venv

# Windows
.\.venv\Scripts\activate

pip install -r requirements.txt
```

## Run

```bash
python main.py
```
